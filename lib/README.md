# Libraries

Third-party libraries are not included in this repository. Clone or download
each one into this `lib/` directory before building.

| Library | Purpose | Source |
|---|---|---|
| HumanCyborgRelationsAPI | HCR vocalizer control (audio + emotions over I2C/serial) | https://github.com/roy86/HumanCyborgRelationsAPI |
| LedControl | MAX7219/MAX7221 LED matrix driver | https://github.com/wayoda/LedControl |
| VarSpeedServo | Variable-speed servo control (originally Michael Margolis) | https://github.com/pvanallen/VarSpeedServo |
